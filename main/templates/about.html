About Edje's Recipe Organizer

This website organizes recipes. It was developed as a training exercise by Edje Jeter, with some inspiration from www.recipe.com.

Key features include:

  • Scaling ingredient quantities to user-determined numbers of servings
  • Calculating nutritional information per serving using data from the USDA's National Nutrition Database (NDB)
  • Allowing users to rate recipes
  • Allowing users to comment on recipes

Remaining tasks include:

  • Setting up "tags"
  • Debugging the servings calculations. (Changing the number of servings and then adding an ingredient causes weirdness.)
  • Improving the UX of adding an ingredient. The list of possible matches is too long and is poorly formatted.
  • Allowing users to submit blocks of ingredients rather than inputting the data one field at a time.
  • Allow users to edit ingredients rather than simply delete them and add another.
  • Improve the formatting, layout, etc, eg, fix the alignment on number of servings, work out dynamic resizing, get the divider lines a little shorter, get directions to have line breaks.
  • Add a search function.
